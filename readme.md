# Đồ án cuối kỳ - Lập trình Web Front-end

## 🌐 Tên đề tài
Website tư vấn và đăng ký trực tuyến các gói cước mạng Cáp quang Internet & Di động - VNPT Digishop.

## 👥 Thành viên nhóm & Phân công công việc
- **Nguyễn Nhựt Thịnh** - MSSV: 26520501002 - *Công việc phụ trách:* Thiết kế Trang chủ / Hero Section, xây dựng hệ thống thanh điều hướng (Navbar) thích ứng responsive, thiết kế chân trang (Footer).
- **Quang Thị Quyền Trang** - MSSV: 26320501004 - *Công việc phụ trách:* Biên soạn nội dung, thu thập dữ liệu gói cước từ hệ thống Digishop, xây dựng phần Giới thiệu, tạo bố cục lưới sản phẩm bằng hệ thống Bootstrap Card.
- **Lê Chí Linh** - MSSV: 26310501014 - *Công việc phụ trách:* Lập trình logic JavaScript, xây dựng thuật toán tìm kiếm kết hợp bộ lọc danh mục nâng cao, viết kịch bản kiểm tra tính hợp lệ dữ liệu (Form Validation), cấu hình lưu trữ GitHub Pages.

## 🛠️ Công nghệ sử dụng
- HTML5 (Cấu trúc ngữ nghĩa chuẩn SEO)
- CSS3 (Đổ màu gradient, hiệu ứng di chuột tương tác mượt mà)
- Thư viện Bootstrap v5.3 (Xử lý Responsive thiết bị & Hệ thống Grid)
- JavaScript ES6 thuần (DOM Manipulation & Xử lý mảng nâng cao)
- GitHub & GitHub Pages (Quản lý mã nguồn nhóm và Hosting triển khai thật)

## ⚡ Các chức năng JavaScript đã hoàn thiện
1. **Tìm kiếm sản phẩm kết hợp bộ lọc danh mục:** Người dùng có thể vừa gõ từ khóa tìm kiếm tên gói cước, vừa lọc sản phẩm theo nhóm (Wifi, Camera, Di động) theo thời gian thực (Real-time).
2. **Kiểm tra dữ liệu Form liên hệ (Form Validation):** Kiểm tra bắt buộc không để trống họ tên, bắt lỗi định dạng email chuẩn xác, ép số điện thoại phải nhập ký số từ 9 - 11 số, giúp ngăn chặn dữ liệu ảo gửi lên máy chủ.
3. **Nút quay về đầu trang (Back to Top):** Nút tròn thông minh ẩn hiện tự động dựa trên tọa độ cuộn màn hình của người dùng, giúp điều hướng trang mượt mà (`smooth scroll`).

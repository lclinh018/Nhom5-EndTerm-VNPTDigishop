Hệ thống tư vấn và đăng ký trực tuyến các gói cước Cáp quang Internet & Di động - VNPT Digishop.

## Công nghệ sử dụng
HTML, CSS, JavaScript, Bootstrap, GitHub Pages

## Chức năng đã làm
1. **Tìm kiếm sản phẩm/dịch vụ kết hợp bộ lọc danh mục nâng cao:** Cho phép lọc các gói cáp quang hoặc sim số di động song song theo thời gian thực.
2. **Kiểm tra dữ liệu Form liên hệ (Form Validation):** Kiểm định nghiêm ngặt cú pháp ký tự email và số điện thoại, hiển thị trạng thái hợp lệ trực quan bằng Bootstrap.
3. **Nút quay về đầu trang (Back to Top):** Nút điều hướng ẩn hiện tự động dựa theo tọa độ cuộn màn hình giúp trải nghiệm lướt mượt mà.
4. **Truyền nhận tham số điều hướng dữ liệu bảng tin (?news=X):** Đọc query string để kết xuất dữ liệu văn bản và hình ảnh tương thích từ trang chủ sang trang Bộ sưu tập.
